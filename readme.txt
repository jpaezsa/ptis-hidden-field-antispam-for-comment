== Installation ==

1. Unpack files from `*.zip` to the `/wp-content/plugins/ptis-hidden-field-antispam-for-comment/` directory or just  "Install" `ptishiddenfield.zip` through admin interface in plugins
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enjoy and time-to-time clean up your spam-comment folder
